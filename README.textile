h1. Why

LaTeX just as any other tool should allow for fluent idea-flow.
When the user becomes interrupted by its semantics, complexity or ...
it might be that he is loosing a great idea in the meantime or for sure 
he loses his time.

h2. Some LaTeX reusable template repository

This repository can be used as a template source for newly created documents and will 
provide source documents just like their output for shortening the way to the proper 
one.
